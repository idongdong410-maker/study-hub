# /set-exam-range 워크플로우

시험 범위를 "data/config.json"에 저장하여, 이후 /generate-study 실행 시 해당 범위 내에서만 컨텐츠를 생성합니다.

## 실행 단계

### 1단계: 사용자에게 시험 범위 질문

사용자에게 아래 정보를 물어봅니다:

`
어떤 과목의 시험 범위를 설정할까요?

1. 미적분 - 예: "미분계수~합성함수 미분"
2. 영어 - 예: "수특 1강~10강"
3. 화법과 작문 - 예: "화법 이론, 토론, 작문 이론"

과목과 범위를 알려주세요. (여러 과목 동시 설정 가능)
`

### 2단계: config.json 업데이트

- "data/config.json" 파일을 엽니다.
- 사용자가 지정한 과목의 "exam_ranges.units" 와 "note"를 업데이트합니다.

예시 파일 형태:
`json
{
  "exam_ranges": {
    "calculus": {
      "units": ["미분계수와 도함수", "접선의 방정식", "합성함수 미분"],
      "note": "중간고사 범위"
    }
  }
}
`

### 3단계: 확인

- 업데이트된 시험 범위를 사용자에게 표 형식으로 명확하게 보여줍니다.
- "이제 /generate-study 실행 시 설정된 이 범위 내에서만 학습 내용과 문제가 생성됩니다." 라고 안내하며 종료합니다.
