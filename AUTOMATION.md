# 자동화 파이프라인 설정 가이드

## 전체 구조
```
Antigravity 에이전트 -> NotebookLM MCP -> JSON 생성 -> GitHub 커밋 -> 앱 자동 동기화
```

## 1단계: 시험 범위 설정 (선택 사항)

과목별 시험 범위가 정해지면 Antigravity 대화창에 아래 명령어를 입력하세요:
`
/set-exam-range
`
그러면 에이전트가 과목과 범위를 확인한 뒤, 이 정보("data/config.json")를 유지합니다.
범위가 설정된 후에는 매일 문제 출제 시 **해당 시험 범위 안에서만** 컨텐츠가 생성됩니다.

---

## 2단계: 일일 학습 자료 자동 생성

원하시는 시간에 컴퓨터에서 Antigravity를 켜시고 채팅창에 딱 한 줄만 입력하세요:
`
/generate-study
`

이 명령어를 실행하면 에이전트가 다음 작업들을 전부 **자동으로 처리**합니다.

1. **미적분** (개념 요약 1개 + 단계별 문제 3개 생성)
2. **영어** (수능특강 4지문 분석 + 지문별 1문제 생성)
3. **화법과 작문** (화작 2지문 분석 + 지문별 1문제 생성)
   *(단, NotebookLM의 지정된 교재 PDF를 바탕으로 구체적인 질문/요약을 수행)*
4. 생성된 데이터를 "data/feed.json" 형식에 맞게 병합 
5. GitHub 레포지토리에 자동 Push

---

## 3단계: 앱에서 확인

Study Hub ("index.html") 앱을 열면 자동으로 가장 최신 상태인 "feed.json"을 읽어옵니다.

- 새 문제 -> StudyFlow
- 새 암기카드 -> MemoryFlow

모든 데이터는 중복 방지 코드가 적용되어 있어 같은 문제가 여러 번 들어가지 않습니다.


## NotebookLM 노트 구조
- **미적분 (2-노트북 분리)**: 수열/급수 등 파트 1과 함수 미분 등 파트 2를 물리적으로 분리하여 NotebookLM 한도를 회피.
- **영어**: 수능특강 1~24강 전체 지문.
- **화법과 작문**: 화법 이론, 작문 이론 등.

## 오답 프롬프트
틀린 문제를 기반으로 NotebookLM에 질문할 때 사용하는 프롬프트:
`[D] 내가 틀린 문제: {문제 내용}. 내 답: {내 답}, 실제 정답: {정답}. 이 문제와 완전히 동일한 논리적 구조를 가지지만 수치만 변경된 변형문제 1개, 그리고 다른 개념과 결합한 심화 변형문제 1개를 만들어줘.`
